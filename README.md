### 领地飞行插件

DomFly 是一个轻量级的 Minecraft 插件，允许玩家在自己宣称的领地内飞行。非常适合服务器经济和建筑导向的游戏玩法！

### ✨ 功能特点

- **领地飞行**: 在领地里输入`/domfly`
- **无缝切换**: 跨越领地边界时平滑启用/禁用飞行
- **经济支持**: 可选的飞行使用费用系统
- **权限控制**: 针对不同用户组的精细权限管理
- **跨插件兼容**: 支持主流领地保护插件
- **自定义消息**: 完全可配置的通知系统

### 📥 安装方法

1. 从发布页面下载最新的 DomFly.jar
2. 将 jar 文件放入服务器的 `plugins` 文件夹
3. 重启服务器
4. 配置插件以匹配您的领地系统
5. 为玩家设置相应权限

### ⚙️ 配置说明

```yaml
# DomFly 配置

```

### 🔐 权限节点

- `domfly.use` - 基础飞行权限
- `domfly.admin` - 管理权限
- `domfly.free` - 绕过经济费用

### 🎮 使用方法

只需进入您自己的领地输入`/domfly`即可开始飞行！插件会自动处理：
- 离开领地时禁用飞行
- 边界检测
